# 🚀 Dropshipping AI Analyzer

An AI-powered web app that helps identify winning dropshipping products using machine learning.

## 🔗 Live Demo
👉 https://your-app-link.streamlit.app

## 📌 Features
- Predict if a product is WINNING or NOT
- Clean and interactive UI using Streamlit
- Uses trained ML model (.pkl)
- Real-time input analysis

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Joblib

## 📊 Input Features
- Views
- Price
- Likes
- Profit Margin
- Comments
- Wow Factor
- Competition

## ⚙️ How to Run Locally

```bash
git clone https://github.com/your-username/dropshipping-ai-app.git
cd dropshipping-ai-app
pip install -r requirements.txt
streamlit run app.py
```
